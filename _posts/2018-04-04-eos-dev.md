### 2. 编译设置开发环境

#### 2.1 获取`EOS`源码及其所有子模块

```
$ git clone https://github.com/eosio/eos --recursive
```

如果克隆代码时未带`--recursive`参数，那么可切换到项目路径下面执行如下命令，更新子模块。

```
$ git submodule update --init --recursive
```

### 2.3 建立区块链节点

切换到目录`eos/build/programs/nodeos`下面，然后执行如下命令。

`./nodeos`命令可附带 `--data-dir 节点文件夹名字` 参数，默认文件夹为`nodeos`。

```
liyuechun:nodeos yuechunli$ ./nodeos 
```
